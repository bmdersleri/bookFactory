# Chapter Structure Standard — Genel Bölüm Yapısı Standardı

Bölüm yapısı `book_manifest.yaml` içindeki `pedagogy.default_chapter_flow` alanından türetilir.

## Varsayılan bölüm akışı

1. Chapter roadmap / Bölümün yol haritası
2. Pedagogical role / Bölümün pedagojik rolü
3. Learning outcomes / Öğrenme çıktıları
4. Prerequisites / Ön bilgi varsayımları
5. Core concepts / Ana kavramlar
6. Step-by-step examples / Adım adım örnekler
7. Working logic / Çalışma mantığı
8. Mini application / Mini uygulama
9. Common mistakes / Sık yapılan hatalar
10. Debugging task / Hata ayıklama egzersizi
11. Summary / Bölüm özeti
12. Glossary / Terim sözlüğü
13. Self-assessment / Kendini değerlendirme
14. Programming exercises / Programlama alıştırmaları
15. Lab/project task / Laboratuvar veya proje görevi
16. Rubric / Değerlendirme rubriği
17. Further reading / İleri okuma
18. Bridge to next chapter / Sonraki bölüme köprü

Başlıklar numarasız üretilir; görünen numaralar build aşamasında atanır.
